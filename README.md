# Railway Leave Management System

## Objective
The objective of this project is to automate the railway employee leave application process by providing a digital platform for leave submission, approval, tracking, and PDF generation with QR code support.

---

## Technologies Used
- Python
- Flask
- Pandas
- HTML
- CSS
- Bootstrap
- ReportLab
- OpenPyXL
- QR Code
- Excel
- Jinja2 Templates

---

## Features
- Employee Login System
- Admin Login Dashboard
- Leave Application Submission
- Leave Status Tracking
- PDF Leave Form Generation
- QR Code Integration
- Public Application Verification
- Leave Approval/Rejection by Admin
- Edit Existing Applications
- Monthly Leave Limit Validation
- Excel-based Data Storage

---

## Leave Types Supported
- On Duty
- LHAP
- LAP
- RH
- CL
- CCL

---

## Workflow
1. User Login
2. Fill Leave Application Form
3. Submit Leave Request
4. Store Application in Excel
5. Generate PDF Application
6. Admin Reviews Application
7. Approve / Reject Application
8. User Tracks Leave Status
